# Step 1: Add container to render IAV flow

Add in a container to the body of your page where you want to render the IAV flow.

```htmlnoselect
<div id="mainContainer">
  <input type="button" id="start" value="Add Bank">
</div>

<div id="iavContainer"></div>
```
